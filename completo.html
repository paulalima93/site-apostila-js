Aula 01 - Variáveis e Tipos Módulo 1

Declaração de Variáveis

let: Variável com escopo de bloco (mais seguro e moderno).

const: Valor constante, não pode ser alterado depois de definido.

var: Forma antiga (pré-ES6), tem escopo de função e pode causar bugs.

Tipos de Dados

  • string (texto)
  • number (inteiro ou decimal)
  • boolean (verdadeiro/falso)

Template Literal

Forma moderna de escrever textos usando crase (` `) e ${}.

// let → variável com escopo de bloco
let idade = 99;

// const → valor constante
const nome = "Paula";

// Exibe nome e idade (concatenação tradicional)
console.log("Nome: " + nome + " | Idade: " + idade);

// Tentar mudar const gera erro:
// nome = "Tais"; // ❌ Erro!

// --- DIFERENÇA VAR, LET, CONST ---
var a = 20;
var b = 20;
var soma = a + b;

if (soma === 40) {
    console.log("É quarenta");
}

let adulto = false;
if (adulto) {
    console.log("Pode votar");
}

// --- EXERCÍCIO: BOATE COM MATINÊ ---
// Regras: 15-17 (matinê), 18+ (boate), <15 (pracinha)
idade = 16;

if (idade >= 15 && idade <= 17) {
    console.log("Olá " + nome + ", pode ir na matinê!");
} else if (idade >= 18) {
    console.log(`Olá ${nome}, pode entrar na boate!`);
} else {
    console.log("Olá " + nome + ", vai para a pracinha.");

// --- TEMPLATE LITERAL ---

console.log(frase);

// --- CURIOSIDADE: VÍRGULA ---
// Retorna apenas a última expressão
const resultado = ("Oi", "Olá", "Bem-vindo!");
console.log(resultado); // Bem-vindo!

Aula 02 - Input e Switch Módulo 1

Entrada de Dados

O módulo readline permite receber dados do usuário pelo terminal.

Funções

São "caixas de ações". Podemos chamar várias vezes com diferentes parâmetros.

Switch Case

Verifica qual valor foi digitado comparando com "casos" (case). Ótimo para menus.

Use break para parar e default para opções inválidas.

const readline = require('readline');

// Interface de leitura
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout
});

// --- FUNÇÕES ---
function comprar(item){
    console.log(`Escolhendo: ${item}`);
    console.log(`Pagando: ${item}`);
    console.log(`Recebendo: ${item}`);
}

let item = "sapato";
comprar(item);

// --- SWITCH E INPUT ---
rl.question('Escolha uma vogal: ', (vogal) => {
  switch (vogal) {
    case "a":
        console.log("Escolheu A");
        break;
    case "e":
        console.log("Escolheu E");
        break;
    case "i":
        console.log("Escolheu I");
        break;
    case "o":
        console.log("Escolheu O");
        break;
    case "u":
        console.log("Escolheu U");
        break;
    default:
        console.log("Não é uma vogal");
        break;
  }
  rl.close();
});

const readline = require('readline')
const rl = readline.createInterface({input: process.stdin, output: process.stdout});

function calculadora(num1, operador, num2) {
    let resultado;
    switch (operador) {
        case "+": resultado = num1 + num2; break
        case "-": resultado = num1 - num2; break
        case "/": resultado = num2 !== 0 ? num1 / num2 : "Erro div 0"; break
        case "*": resultado = num1 * num2; break
        default: resultado = 'Operação inválida'
    }
    return resultado
}

rl.question("Num 1: ", (n1) => {

      console.log("Resultado:", calculadora(parseFloat(n1), op, parseFloat(n2))) 
      rl.close();
    });
  });
});

Aula 04 - Operadores Módulo 1

Matemáticos

Soma, subtração, multiplicação, divisão, resto (%) e potência (**).

Lógicos

  • && (AND): Tudo true.
  • || (OR): Pelo menos um true.
  • ! (NOT): Inverte.

Relacionais

Comparação de valores. Atenção ao === que compara valor E tipo.

// --- MATEMÁTICOS ---
let a = 9, b = 12;

console.log("Soma:", a + b);
console.log("Subtração:", b - a);
console.log("Multiplicação:", a * b);
console.log("Divisão:", b / a);
console.log("Resto:", b % a); // 3
console.log("Potência:", a ** b); 

// --- LÓGICOS ---
let temCNH = false;
let temRG = true;

// AND (&&)
if (temCNH && temRG) console.log("Docs válidos");

// OR (||)
if (temCNH || temRG) console.log("Tem algum doc");

// NOT (!)
if (!temCNH) console.log("NÃO tem CNH");

// --- RELACIONAIS ---
// == valor, === valor e tipo
let comparar = 5 === "5"; // false (number vs string)
console.log("5 === '5'?", comparar);

if (a % 2 === 0) console.log(a + " é par");
else console.log(a + " é ímpar");

Aula 05 - Laços de Repetição Módulo 1

FOR (Controle Fixo)

Executa um número conhecido de vezes.
Estrutura: (início; condição; passo).

WHILE (Enquanto)

Executa enquanto a condição for verdadeira. Ideal quando não sabemos o fim.

DO...WHILE

Executa primeiro, verifica depois. Garante pelo menos uma execução.

// --- FOR ---
// De 0 a 4
for (let i = 0; i < 5; i++) {
  console.log("Número: " + i);
}

// De 1 a 6 (inclusive)
for (let i = 1; i <= 6; i++) {
  console.log("Número: " + i);
}

console.log("===");

// --- WHILE ---
let count = 0;

while (count < 5) {
  console.log("Contando: " + count);
  count++;
}

console.log("===");

// --- DO...WHILE ---
let check = 10;

do {
  console.log("Checando: " + check);
  check++;
} while (check < 5);
// Executa 1 vez mesmo sendo falso (10 não é < 5)

Padrão para buscar, validar ou substituir texto.

Símbolos

Aula 06 - Try / Catch Módulo 1

Tratamento de Erros

Impede que o código quebre completamente ao encontrar um erro.

  • try: Código que pode falhar.
  • catch: Captura e trata o erro.
  • throw: Lança um erro manual.

Útil para validações, parse de JSON e acesso a objetos nulos.

// 1. Objeto nulo
const usuario = null;
try {
    console.log(usuario.nome);
} catch (erro) {
    console.warn("Erro ao acessar:", erro.message);
}

// 2. JSON Inválido
const jsonInvalido = "{ nome: 'João' }"; // Falta aspas
try {
    const obj = JSON.parse(jsonInvalido);
} catch (erro) {
    console.error("Erro JSON:", erro.message);
}

// 3. Erro Manual (Throw)
function dividir(a, b) {
    if (typeof a !== "number") throw new Error("Apenas números");
    if (b === 0) throw new Error("Divisão por zero");
    return a / b;
}

try {
    dividir(10, 0);
} catch (erro) {
    console.error("Erro dividir:", erro.message);
}

// 4. Regex + Try/Catch
function validarEmail(email) {
    const regexEmail = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    if (!regexEmail.test(email)) throw new Error("E-mail inválido!");
    return true;
}

Aula 07 - Arrays Módulo 1

Listas Ordenadas

Armazenam vários valores. Índice começa em 0.

Métodos Principais

  • push/pop: Fim.
  • unshift/shift: Início.
  • includes/indexOf: Busca.
  • slice: Cortar.
  • splice: Remover/Alterar.
  • join: Virar string.

Loops em Arrays

Percorrer para somar, filtrar ou exibir dados.

let frutas = ["maçã", "banana", "uva", "laranja"];
console.log(frutas[0]); // maçã

// Adicionar/Remover
frutas.push("melancia"); // Fim
frutas.pop(); 
frutas.shift(); // Remove do início
frutas.unshift("abacaxi"); // Adiciona no início

// Busca
console.log(frutas.includes("uva")); // true
console.log(frutas.indexOf("banana")); // Índice

// Manipulação
let favoritas = frutas.slice(1, 3); // Cópia parcial
frutas.splice(1, 2); // Remove 2 a partir do índice 1
console.log(frutas.join(", ")); // String única
frutas.reverse();
frutas.sort();

// --- LOOPS ---
let nomes = ["Ana", "Carlos", "Julia"];
for (let i = 0; i < nomes.length; i++) {
  console.log("Nome:", nomes[i]);
}

// Soma
let numeros = [10, 20, 30, 40];
let soma = 0;
for (let i = 0; i < numeros.length; i++) {
  soma += numeros[i];

Aula 08 - Funções Avançadas Módulo 1

Conceitos

Funções reutilizam código. Nomeie com verbos (ex: calcular).

Parâmetros Padrão: Evita "undefined" definindo valores default.

Retorno: return devolve valor e encerra a função.

Callbacks: Funções passadas como argumento para rodar depois (síncrona ou assíncrona).

Arrow Functions: Sintaxe curta (n) => n * 2.

// Declaração
function saudacao() {
  console.log("Olá! Bem-vindo.");
}
saudacao();

// Parâmetros Default
function apresentar(nome, idade = 0) {
  console.log(`Sou ${nome}, ${idade} anos.`);
}
apresentar("Paula", 98);
apresentar("João"); // idade = 0

// Retorno (Objeto)
function dividir(dividendo, divisor) {
  if (divisor === 0) return { ok: false, erro: "Divisão por zero" };
  return { ok: true, res: dividendo / divisor };
}
console.log(dividir(17, 5));

// Expressão e Arrow
const dobrar = function (n) { return n * 2; };
const triplicar = (n) => n * 3;

// Callbacks Síncronos
function executar(a, b, operacao) {
  return operacao(a, b);
}
console.log(executar(10, 5, (x, y) => x - y)); // 5

// Callback Assíncrono (Timer)
console.log("Início");
setTimeout(() => {
  console.log("Rodou após 500ms");
}, 500);
console.log("Fim");

Aula 09 - JSON Módulo 1

O que é JSON?

Formato de texto para dados. Muito usado em APIs.

  • Chaves entre aspas duplas.
  • Valores: strings, números, arrays, objetos, booleans.
  • Não aceita funções ou undefined.

Manipulação

JSON.parse(): Transforma texto em objeto (para usar no código).

JSON.stringify(): Transforma objeto em texto (para enviar/salvar).

// String JSON
const pessoaJSON = `{
  "nome": "Paula",
  "idade": 98,
  "ativo": true,
  "cursos": ["JS", "Python"]
}`;

// Parse (Texto -> Objeto)
const pessoaObj = JSON.parse(pessoaJSON);
console.log(pessoaObj.nome); // "Paula"
pessoaObj.idade += 1; // Podemos editar agora

// Stringify (Objeto -> Texto)
const objJS = { nome: "João", idade: 35 };
const objJSON = JSON.stringify(objJS);
console.log(objJSON); 
// '{"nome":"João","idade":35}'

// Arrays e Objetos Aninhados
const dadosJSON = `{
  "usuarios": [
    { "nome": "Ana", "idade": 22 },
    { "nome": "Carlos", "idade": 30 }
  ]
}`;
const dadosObj = JSON.parse(dadosJSON);
console.log(dadosObj.usuarios[0].nome); // "Ana"

Conceito

Combinam ou invertem condições booleanas.

  • || (OR): Pelo menos uma verdadeira.

const idade = 20;

console.log(idade < 18 || possuiCarteira); // true

// NOT (!)

Aula 11 - Métodos de String Módulo 1

Manipulação de Texto

Strings são imutáveis (métodos retornam nova string).

Métodos Comuns

  • length: Tamanho.
  • toUpperCase/Lower: Caixa alta/baixa.
  • includes: Verifica existência.
  • replace: Substitui.
  • substring: Pega um pedaço.
  • split: Divide em array.
const nome = "Paula Lima";

// Tamanho
console.log(nome.length); // 10

// Maiúsculas/Minúsculas
console.log(nome.toUpperCase()); // "PAULA LIMA"
console.log(nome.toLowerCase()); // "paula lima"

// Procurar e Substituir
console.log(nome.includes("Paula")); // true
console.log(nome.replace("Paula", "Joana")); // "Joana Lima"

// Cortar e Dividir
console.log(nome.substring(0, 5)); // "Paula"
console.log(nome.split(" ")); // ["Paula", "Lima"]

// Concatenar
console.log(`${nome}, seja bem-vinda!`);

Aula 12 - Objeto Math Módulo 1

Matemática

Objeto global para cálculos. Não precisa instanciar.

Métodos Úteis

  • round/floor/ceil: Arredondamento.
  • random: Número aleatório (0 a 1).
  • pow/sqrt: Potência e Raiz.
  • abs: Valor absoluto (sem sinal).
  • min/max: Mínimo e Máximo.
console.log(Math.PI);     // 3.1415...

// Arredondamento
console.log(Math.round(4.7)); // 5
console.log(Math.floor(4.7)); // 4 (Chão)
console.log(Math.ceil(4.2));  // 5 (Teto)

// Aleatórios
console.log(Math.random()); 
console.log(Math.floor(Math.random() * 10)); // 0 a 9

// Potência e Raiz
console.log(Math.pow(2, 3)); // 8
console.log(Math.sqrt(25));  // 5

// Absoluto
console.log(Math.abs(-10)); // 10

// Limites
console.log(Math.max(3, 7, 1)); // 7
console.log(Math.min(3, 7, 1)); // 1
